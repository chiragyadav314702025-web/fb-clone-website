This is my first GitHub repository, created while learning Git and GitHub.

The purpose of this project is to understand the basic concepts of:

- Git initialization
- Cloning a repository

- Resolving merge conflicts
- Pushing changes to GitHub
- Working with remote repositories

This repository is a part of my Git and GitHub practice. I am using it to learn how version control works and how developers collaborate on projects.
